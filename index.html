🗺 What is LuaU?

Luau is Roblox's open-source, gradually-typed programming language — a fast, safe superset of Lua 5.1. It adds a powerful type system, better performance, and new syntax features while remaining fully backward-compatible.

All scripts in Roblox run on Luau. Understanding it deeply is the difference between a hobbyist game and a professional, performant experience.

Typed Lua Lua 5.1 compatible Open Source Gradual Types No Metatables for Userdata
⚡ Key Luau differences from Lua 5.1
Type annotations▶
Luau allows local x: number = 5 — gradual, optional types that can be enforced by the type checker. This catches bugs before runtime.
Generalized iteration▶
In Luau, you can write for k, v in t do — no need for pairs() or ipairs(). The runtime figures it out automatically.
String interpolation▶
Luau supports `Hello, {name}!` — backtick strings with embedded expressions using {}.
Continue statement▶
Luau adds continue inside loops — skips the rest of the loop body and moves to the next iteration. Not in standard Lua 5.1.
floor division operator▶
Luau includes // for integer floor division, e.g., 7 // 2 == 3.
Buffer type▶
Luau has a buffer type for fixed-size binary data, useful for serialization and networking.

📌 Declaring Variables

Luau has three scoping keywords. Always prefer local — it's faster and avoids polluting the global environment.

luau
local x = 10              -- local: block-scoped (preferred)

local a, b = 1, 2         -- multiple assignment
local c                   -- nil by default

-- Luau type annotations (optional but recommended)
local name: string = "Zeeval"
local score: number = 100
local alive: boolean = true

Common Gotcha

Luau does not have ternary ? : syntax. The idiom is: condition and valueIfTrue or valueIfFalse. But be careful: if valueIfTrue is falsy, this breaks! Use an if/else function for safety.

while loop▶

    print(i)

repeat...until▶

Runs at least once. Condition is checked after each iteration.

luau
local n = 0
repeat
    n = n + 1
until n >= 5
print(n)  -- 5
Numeric for▶
luau
-- for var = start, stop [, step] do
for i = 1, 10 do           -- 1 to 10, step 1
    print(i)
end

for i = 10, 1, -2 do      -- 10, 8, 6, 4, 2
    print(i)
end
Generic for (ipairs / pairs)▶
luau
local fruits = {"apple", "banana", "cherry"}

-- ipairs: iterate array (1, 2, 3...)
for i, v in ipairs(fruits) do
    print(i, v)  -- 1 apple, 2 banana, 3 cherry
end

local stats = { kills=10, deaths=3 }
-- pairs: iterate all keys (order not guaranteed)
for k, v in pairs(stats) do
    print(k, v)
end

-- Luau generalized iteration (no pairs/ipairs needed!)
for k, v in stats do
    print(k, v)
end
⏭ break & continue (Luau)
luau
for i = 1, 10 do
    if i == 3 then
        continue    -- skip 3 (Luau only!)
    end
    if i == 7 then
        break       -- stop at 7
    end
    print(i)       -- prints 1, 2, 4, 5, 6
end
Luau Exclusive

continue does not exist in standard Lua 5.1. It's a Luau addition that skips to the next loop iteration.

📌 Declaring Functions

    end
    return b, a
end

📦 Varargs (...)
luau
local function sum(...)
    local args = {...}  -- pack into table
    local total = 0
    for _, v in args do
        total += v     -- compound assignment (Luau!)
    end
    return total
end

print(sum(1, 2, 3, 4))  -- 10

-- table.pack and table.unpack
local function test(...)
    local packed = table.pack(...)
    print(packed.n)  -- packed.n = arg count
end
Luau Compound Assignment

Luau supports +=, -=, *=, /=, //=, %=, ^=, ..= — these are compound assignment operators not in standard Lua 5.1.

🔒 Closures

A closure is a function that captures variables from its enclosing scope (upvalues).

luau
local function makeCounter(start: number)
    local count = start  -- upvalue captured by closure
    return function()
        count += 1
        return count
    end
end

local counter = makeCounter(0)
print(counter())  -- 1
print(counter())  -- 2
print(counter())  -- 3
📐 Method Syntax (: vs .)
luau
local obj = { name = "Hero" }

-- Using . — you must pass self manually
function obj.greet(self)
    print("I am " .. self.name)
end
obj.greet(obj)  -- manual self

-- Using : — self is implicit (syntactic sugar)
function obj:greet2()
    print("I am " .. self.name)
end
obj:greet2()   -- self injected automatically
Rule of Thumb

Use : for methods (functions that operate on an object). Use . for regular functions stored in tables.

Metatables
Control how tables behave with metamethods
🔑 What are Metatables?

A metatable is a regular table that defines special behavior for another table. Metamethods are keys starting with __ that hook into Lua operations.

luau
local t = {}
local mt = {
    __index = function(tbl, key)
        return "default"  -- return default for missing keys
    end
}
setmetatable(t, mt)

print(t.anything)  -- "default" (key doesn't exist!)
t.name = "real"
print(t.name)     -- "real" (__index not called)

🧮 Vector2 example with metamethods
luau
local Vec2 = {}
Vec2.__index = Vec2

function Vec2.new(x, y)
    return setmetatable({x=x, y=y}, Vec2)
end

function Vec2.__add(a, b)
    return Vec2.new(a.x + b.x, a.y + b.y)
end

function Vec2.__tostring(v)
    return string.format("(%g, %g)", v.x, v.y)
end

local a = Vec2.new(1, 2)
local b = Vec2.new(3, 4)
local c = a + b              -- calls __add
print(tostring(c))         -- "(4, 6)" via __tostring

Object-Oriented Programming
Classes, inheritance, and encapsulation in Luau
🏗️ The Class Pattern
luau
local Animal = {}
Animal.__index = Animal  -- instances look up methods here

-- Constructor
function Animal.new(name: string, sound: string)
    local self = setmetatable({}, Animal)
    self.name  = name
    self.sound = sound
    return self
end

-- Method
function Animal:speak()
    print(`{self.name} says {self.sound}!`)
end

function Animal:getName(): string
    return self.name
end

local cat = Animal.new("Cat", "Meow")
cat:speak()  -- "Cat says Meow!"
🧬 Inheritance
luau
local Dog = setmetatable({}, {__index = Animal})
Dog.__index = Dog

function Dog.new(name: string)
    local self = Animal.new(name, "Woof")
    return setmetatable(self, Dog)
end

-- Override method
function Dog:fetch()
    print(`{self.name} fetches the ball!`)
end

local rex = Dog.new("Rex")
rex:speak()   -- inherited: "Rex says Woof!"
rex:fetch()   -- "Rex fetches the ball!"
Lookup chain

When rex:speak() is called: Lua looks in rex (not found) → looks in Dog (not found) → looks in Animal (found!). This is the prototype chain via __index.

QUIZ

BTo make Animal callable as a constructor
CSo that instances look up missing keys in the Animal table
DIt's just convention, it has no effect
✓ Correct! When an instance doesn't have a key, Lua checks its metatable's __index. By pointing that to Animal, all instances share the same methods.
✗ Animal.__index = Animal means: when an instance is missing a key, look it up in Animal — this is how all instances share class methods.

Coroutines
Cooperative multitasking with coroutine library
⚙️ Coroutine Basics

Coroutines are functions that can be paused and resumed. They're the foundation of task.spawn and wait() in Roblox.

luau
local function producer()
    for i = 1, 3 do
        print("Producing " .. i)
        coroutine.yield(i)   -- pause and send value
    end
    return "done"
end

local co = coroutine.create(producer)

local ok, v = coroutine.resume(co)  -- "Producing 1", v=1
ok, v = coroutine.resume(co)       -- "Producing 2", v=2
ok, v = coroutine.resume(co)       -- "Producing 3", v=3
ok, v = coroutine.resume(co)       -- v="done", coroutine ends

📦 buffer library (Luau)

The buffer type is a fixed-size binary blob, useful for efficient binary serialization.

luau
local b = buffer.create(8)     -- 8 bytes

buffer.writei32(b, 0, 42)      -- write int32 at offset 0
buffer.writef32(b, 4, 3.14)    -- write float32 at offset 4

local n = buffer.readi32(b, 0)   -- 42

⏱ task library

The task library is Roblox's scheduler API. It is the modern replacement for old throttled timing patterns, and it works with the engine's scheduler.

luau

luau
